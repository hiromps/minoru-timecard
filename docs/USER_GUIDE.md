# ミノルタイムカードシステム 従業員向け 打刻マニュアル

このマニュアルは、ミノル化学工業株式会社の勤怠打刻システム「ミノルタイムカードシステム」で、
従業員のみなさんが出勤・退勤の打刻を行うための操作手順書です。

パソコンやスマートフォンの操作が得意でない方でも迷わないよう、画面の表示どおりに手順を記載しています。

---

## 1. 打刻画面を開く

1. 会社から案内されたURL（打刻用アドレス）をブラウザで開きます。
   URLの末尾は「`/`」（トップページ）です。管理者用の `/admin` は使いません。
2. 画面上部に「**現在時刻**」と、その下に現在の日時が1秒ごとに更新されて表示されます。
   この時刻が正しく表示されていれば、打刻の準備は完了です。

> このシステムは社内ネットワーク内での利用を前提としています。画面が開けない場合は、
> ネットワーク接続を確認のうえ、管理者（人事担当）にお問い合わせください。

---

## 2. 自分の名前（社員）を選ぶ

打刻の前に、必ず自分を選択します。

1. 「**社員選択**」の下にあるドロップダウン（「社員を選択してください」と表示されている欄）をクリックします。
2. 一覧から「**社員ID - 氏名**」（例: `001 - 田中太郎`）の形式で表示される、自分の名前を選びます。

社員を選ぶと、その下に次の情報が表示されます。

- **本日の状況**（すでに打刻がある場合）: 本日の日付、ステータス、出勤・退勤時刻、勤務時間
- **出勤 / 退勤** の2つのボタン
- **その月の勤務記録**（折りたたみ表示）

> 一覧に自分の名前が無い場合は、まだ社員登録がされていません。管理者にご連絡ください。

---

## 3. 出勤の打刻

1. 自分の名前を選択します（第2章）。
2. 緑色の「**出勤**」ボタンを押します。
3. 確認画面（ポップアップ）が表示されます。内容は次のとおりです。
   - 「〇〇さんの**出勤**を記録します」というメッセージ
   - 打刻時刻の選び方（初期は「**現在時刻で打刻**」が選ばれています）
4. そのまま今の時刻で打刻する場合は、「**確認**」ボタンを押します。
5. 「**出勤打刻が完了しました**」と表示されれば完了です。

打刻が終わると、社員の選択は自動的に解除されます。連続して別の人が打刻する場合に便利です。

---

## 4. 退勤の打刻

1. 自分の名前を選択します。
2. 赤色の「**退勤**」ボタンを押します。
3. 確認画面で「〇〇さんの**退勤**を記録します」と表示されます。
4. 「**現在時刻で打刻**」が選ばれていることを確認し、「**確認**」ボタンを押します。
5. 「**退勤打刻が完了しました**」と表示されれば完了です。

退勤打刻をすると、その日の**勤務時間**が自動計算されます（昼休憩 12:00〜13:00 に重なる時間は自動で差し引かれます）。

> **退勤は出勤より後の時刻でなければ記録できません。** 出勤打刻をせずに退勤しようとした場合や、
> 出勤より前の時刻を指定した場合はエラーになります。

---

## 5. 直行・直帰モードで打刻する

外回りや出張などで、会社の始業時刻・終業時刻どおりに打刻できない場合に使うモードです。
このモードで打刻すると、**遅刻・早退・残業の判定を行わず「通常」扱い**になります（労働時間はきちんと計上されます）。

直行・直帰モードは、**時刻を指定して打刻するとき**のみ選べます。

1. 自分の名前を選択し、「**出勤**」または「**退勤**」ボタンを押します。
2. 確認画面で「**時刻を指定**」を選びます（下記・第6章参照）。
3. 時刻の入力欄の下に表示される、
   「**直行・直帰モード（遅刻・早退判定を無効化）**」のチェックボックスにチェックを入れます。
4. 時刻を入力し、「**確認**」ボタンを押します。
5. 完了メッセージに「**（直行・直帰）**」と付いて表示されれば、直行・直帰として記録されています。

> 直行・直帰かどうかの判断や運用ルールは、会社の規定に従ってください。不明な場合は管理者に確認しましょう。

---

## 6. 時刻を指定して打刻する（後から入力する場合）

打刻を忘れていて後から入力したい場合や、実際の出勤・退勤時刻で記録したい場合に使います。

1. 自分の名前を選択し、「**出勤**」または「**退勤**」ボタンを押します。
2. 確認画面で「**時刻を指定**」のラジオボタンを選びます。
3. 表示された時刻入力欄に、記録したい時刻を「**時:分**」で入力します
   （初期値として現在時刻が入っています）。
4. 必要であれば「直行・直帰モード」にチェックを入れます（第5章）。
5. 「**確認**」ボタンを押します。

> - 指定できるのは**当日の時刻**です。前日以前の打刻もれは、システムでは修正できないため、
>   管理者に依頼して修正してもらってください（管理者マニュアル参照）。
> - 退勤時刻は出勤時刻より後にしてください。24時間を超える勤務時間は登録できません。

---

## 7. 自分の当日の記録・月間カレンダーを確認する

### 本日の状況

自分の名前を選ぶと、その日にすでに打刻があれば「**〇〇さんの本日の状況**」カードが表示されます。

- **ステータス**バッジ（例: ✅ 通常 / ⚠️ 遅刻 / 🏃 早退 / 💪 残業 / ⚠️🏃 遅刻・早退 / ⚠️💪 遅刻・残業）
- 🌅 出勤時刻、🌙 退勤時刻、⏰ 勤務時間

### その月の勤務記録（カレンダー）

1. 自分の名前を選択します。
2. 「**〇〇年〇月の勤務記録**」という見出しの行をクリックすると、記録一覧が開きます（もう一度押すと閉じます）。
3. 日付ごとに、ステータス・出勤時刻・退勤時刻・勤務時間が一覧で確認できます。

まだ打刻が無い月は「**記録がありません**」と表示されます。

#### ステータスの意味

| 表示 | 意味 |
|------|------|
| ✅ 通常 | 始業〜終業の時間どおりに勤務 |
| ⚠️ 遅刻 | 出勤時刻が所定の始業時刻より遅い |
| 🏃 早退 | 退勤時刻が所定の終業時刻より早い |
| 💪 残業 | 退勤時刻が所定の終業時刻を超えている |
| ⚠️🏃 遅刻・早退 | 遅刻かつ早退 |
| ⚠️💪 遅刻・残業 | 遅刻かつ残業 |

> 表示が実態と違う場合や「設定エラー」と出ている場合は、管理者にご相談ください。

## 8. よくある質問（FAQ）

### Q1. 打刻を忘れてしまいました。どうすればよいですか？

- **当日中に気づいた場合**: 「時刻を指定して打刻」（第6章）で、実際の出勤・退勤時刻を入力してください。
- **翌日以降に気づいた場合**: 従業員の画面からは過去の日付を修正できません。
  管理者（人事担当）に、日付・正しい時刻・理由を伝えて修正を依頼してください。

### Q2. 出勤ボタンを2回押してしまいました（二重打刻）

**1日1回分の記録**しか作られない仕組みのため、同じ日に何度も打刻しても記録が増えて二重になることはありません。
時刻がおかしくなってしまった場合は、管理者に修正を依頼してください。

### Q3. 退勤ボタンを押したらエラーになりました

退勤時刻が出勤時刻より前になっていないか確認してください。出勤打刻がまだの場合は、
先に出勤を記録してから退勤してください。解決しない場合は管理者にご連絡ください。

### Q4. 自分の名前が一覧に出てきません

社員登録がまだされていない可能性があります。管理者に登録を依頼してください。

### Q5. 打刻した時刻が実際とずれています

システムは日本時間（JST）で表示・記録します。もし大きくずれている場合は、
使用している端末の時刻設定を確認し、それでも直らない場合は管理者にご連絡ください。
